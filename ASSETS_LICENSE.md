# Assets

`public/assets/` のPNG画像は本プロジェクト向けにAI生成した独自アセット、`bgm_village.ogg` は合成音だけで生成したオリジナル音源です。MITライセンスのコードとは分離し、法令および利用した生成サービスの規約に従う範囲で、本プロジェクトとともに利用・改変・再配布できます。特定の人狼ゲーム製品、企業、団体の素材ではありません。

`sfx_scene_change.ogg`、`sfx_vote.ogg`、`sfx_attack.ogg`、`sfx_execution.ogg` はKenneyの「UI Audio」「RPG Audio」収録素材です。配布元の宣言および同梱ライセンスに基づき、Creative Commons Zero（CC0 1.0 Universal）として利用しています。クレジットは必須ではありませんが、出典の追跡性のため記録しています。

- UI Audio: https://kenney.nl/assets/ui-audio
- RPG Audio: https://kenney.nl/assets/rpg-audio
